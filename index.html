¡Bienvenido a Vim!

En este primer nivel aprenderás a abrir Vim, comprender sus modos y los comandos básicos de cada uno.

Básico Intermedio Avanzado

Nivel 1: Paso a paso

1. Abrir la terminal y Vim

En tu sistema (Linux/Mac/Windows WSL), abre la terminal y escribe:

vim nombre_de_archivo.txt

Esto abrirá Vim en modo Normal.

2. Comprender los modos de Vim

Vim es un editor modal, lo que significa que cambia de comportamiento según el modo en el que estés:

  • Normal: Navegación, borrado y comandos de control.
  • Insert: Inserción de texto.
  • Visual: Selección de bloques de texto.
  • Command-line: Ejecutar órdenes como guardar o buscar.

3. Cambiar entre modos

Acción De A Comando
Insertar texto Normal Insert i
Seleccionar texto Normal Visual v
Ejecutar comando Normal Command-line :
Volver a Normal Insert/Visual Normal Esc

4. Comandos básicos en cada modo

Normal

Acción Comando
Mover cursor h, j, k, l
Borrar carácter x
Deshacer u
Repetir última acción .

Insert

Entra en modo Insert con i. Escribe texto como en cualquier editor. Sal de Insert con Esc.

Visual

Inicia selección con v. Muévete con h/j/k/l. Copia con y, corta con d. Sal de Visual con Esc.

Command-line

Entra con :. Guarda con :w, sale con :q, guarda y sale con :wq o ZZ.

5. Ejercicio práctico

Crear y editar un archivo
  1. Abre Vim: vim prueba.txt.
  2. Entra en Insert (i) y escribe varias líneas.
  3. Sal a Normal (Esc), mueve el cursor y borra una letra (x).
  4. Selecciona una palabra en Visual (v + mover), cópiala (y) y pégala (p).
  5. Guarda y sal (:wq).
